# Free Speech Survey Analysis

**URL**: https://github.com/milesbarry/free-speech-analysis

Analysis of student attitudes toward free speech on campus and social media platforms, conducted as part of UC VOICE-funded research with Dr. Jens Pohlmann.

## Overview

This project examines self-censorship patterns among college students using survey data and statistical analysis in R.

## Methods

- Qualtrics survey design
- R programming for data cleaning and visualization
- Multi-response question analysis
- ggplot2 and plotly for interactive visualizations

## Key Findings

Initial analysis reveals interesting patterns around generational differences in speech expectations and platform-specific norms.
